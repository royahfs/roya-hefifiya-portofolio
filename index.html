About Me

I’m a software developer with a background in Information Systems who loves turning ideas into real digital products. I work on both web and Android development, focusing on clean logic, smooth user experience, and solutions that actually work.

I enjoy building responsive websites, developing Android apps using Java, and continuously learning new technologies. For me, coding isn’t just about making things run — it’s about creating something useful, simple, and impactful.

Skill

UI Design

I create user interface designs using Figma to visualize layouts and components before implementation.
My focus is on clean layout, visual consistency, and preparing designs that are ready to be translated into web or mobile interfaces.

Web Development

I develop dynamic web applications using HTML, CSS, JavaScript, and PHP. I have experience building web projects with the CodeIgniter framework, handling both frontend interfaces and server-side logic to create structured and functional web systems.

Android Development

I build Android applications using Java with a focus on application flow, core functionality, and user interaction. I aim to create Android apps that are stable, efficient, and easy to use.
